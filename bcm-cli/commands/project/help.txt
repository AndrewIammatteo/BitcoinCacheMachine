Description:
  Manage BCM projects.

Usage:
  bcm project [command] [projectname]

Available Commands:
  create        Create BCM project.
  destroy       Destroy BCM project (deletes ~/.bcm/projects/<projectname>).
  list          List all bcm projects.
  get-default   Shows active BCM project.
  set-default   Sets BCM active project.

  Use "bcm project [command]" to get help for that command.
