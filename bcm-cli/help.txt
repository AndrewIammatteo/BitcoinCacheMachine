Description:
  Command line client for Bitcoin Cache Machine.

  All of BCM's features can be driven through the various commands below.

Usage:
  bcm [command]

Available Commands:
  project       Manage BCM projects.
  cluster       Create, destroy, and manage BCM clusters (i.e., LXD cluster)
  git           Perform typical git operations with a hardware wallet backend.

Help
  Use "bcm [command]" to get help for that command.
