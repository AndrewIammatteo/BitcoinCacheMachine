Description:
  Generate a new SSH username@hostname keypair backed by your Trezor.

Usage:
  bcm ssh newkey --username=<BCM_SSH_USERNAME> --hostname=<BCM_SSH_HOSTNAME> --push

Required Parameters:
  --username   The username to use in the SSH username/hostname identifier.

  --hostname   The fully qualified DNS-resolveable name of the host the key is meant for.
                                      (A UNIQUE SSH key MUST be generated for each username/hostname pair)

Optional:
  --push           If specified, the new SSH public key will be pushed to the destination via 'ssh-copy-id'.
