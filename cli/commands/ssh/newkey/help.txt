Description:
  Generate a new SSH username@hostname keypair backed by your Trezor.

Usage:
  bcm ssh newkey [--cluster-name=$BCM_CLUSTER_NAME] --endpoint-name=$BCM_ENDPOINT_NAME [--push]

Parameters:
  --cluster-name=   Optional. Specify the BCM cluster the endpoint is a member of.
  --endpoint-name=  Required. The name of the endpoint. The active BCM_CLUSTER_NAME is assumed.

Optional:
  --push        If specified, the new SSH public key will be pushed to the destination via 
                ssh-copy-id (torified if hostname is a .onion site).
